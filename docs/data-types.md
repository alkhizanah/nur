# Syphon's Data Types

## Number

- Integer

A 64-bit number with no floating-points

```
7
-19
-90
```

- Float

A 64-bit floating-points number

```
0.5
55.0
-1.5
200.19
```

## String

A list of characters

```
"Hello, World!"
```

## Boolean

A true or A false

```
true
false
```

## Array

A list of values

```
[0, 1, "two", false, 4]
```

## None

A value represents nothing

```
none
```

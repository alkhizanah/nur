# The Mathematic Built-in Module

## Expectations

I expect that you imported the math module like that

```
math = import("math")
```

## Constants

- pi
- e
- tau
- phi

## Functions

- sqrt

Gives you an approximation of the square root of a number

```
math.sqrt(4) # 2.0
math.sqrt(65) # 8.06225774829855
```

# Syphon's Syntax

## Variable Declaration

- Mutable Variable
```
let variable_name = variable_value
```

- Unmutable Variable (Constants)
```
const variable_name = variable_value
```

- None-Initialized Variable
```
let variable_name;
```

## Function Declaration

- No Parameters
```
fn function_name() {

}
```

- With Parameters
```
fn function_name(a, b) {

}
```

## Function Calling

- No Arguments
```
function_name()
```

- With Arguments
```
function_name(1, 2)
```

if condition {

}
